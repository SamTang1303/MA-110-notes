# Problem 2.23 MA-110

> [!problem] Problem 2.23
> Translate each of the following into additive notation:
> 1. $a^{n}=e$
> 2. "There exists an element $x$ such that $ax=b^{-1}$."
> 3. $a^{-1}ba=e$
> 4. $(a^{-1})^{n}=(a^{n})^{-1}$
> 5. $(a^{-1})^{-1}=a$
> 6. $a^{n}a^{m}=a^{n+m}$
> 7. $(a^{n})^{m}=a^{nm}$

1. $na=0$.
2. "There exists an element $x$ such that $a+x=-b$."
3. $-a + b + a=0$.
4. $n(-a) = -(na)$.
5. $-(-a)=a$.
6. $na+ma=(n+m)a$.
7. $m(na) = (mn)a$.
